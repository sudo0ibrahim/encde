# encde
Simplified word coding / تشفير مبسط للكلمات

The encryption key is any number / مفتاح التشفير أي رقم

## Example / مثال
encryption key in the example 19 /  19 مفتاح التشفير في المثال
```batch
encode[e] decode[d]: e
word: hello
key: 19
axeeh

encode[e] decode[d]: d
word: axeeh
key: 19
hello

```
